# WorkPace Design System

Welcome to the WorkPace Design System. This is a collection of reusable components and guidelines for building consistent and accessible user interfaces.

If you are new around here, checkout the Getting Started section below to learn how to use the components in your project.

## Getting Started

This guide will get you setup and running with the Workpace Design System!

### Installation
To install the npm package, run the following command:

```shell
npm install @work-pace/design-system
```
